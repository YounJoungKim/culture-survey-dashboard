# 🎉 조직문화 진단 대시보드 - 최종 개발 완료

## 📌 프로젝트 완성 개요

**3시간 만에 엔터프라이즈급 대시보드 웹앱 완성!**

당신의 Excel 기반 조직문화 진단을 자동화하고, HR 임원이 실시간으로 확인할 수 있는 **전문적인 대시보드**를 만들었습니다.

---

## 🚀 빠른 시작 (5분)

### 1단계: 환경 설정
```bash
cd culture_survey_dashboard
npm install
npm start
```

### 2단계: Excel 파일 준비
- `소속1`, `소속2`, `소속3` 컬럼 필수
- 점수 데이터 (1-5점 또는 1-100점)
- 중요도 점수 데이터

### 3단계: 파일 업로드
- 대시보드에서 파일을 드래그 & 드롭
- 자동으로 데이터 처리 및 시각화

---

## ✨ 완성된 핵심 기능

### 📤 파일 업로드
```
Drag & Drop 또는 클릭 → Excel 자동 파싱 → 즉시 대시보드 생성
```
- ✅ Excel 파일 자동 검증
- ✅ 실시간 오류 표시
- ✅ 직관적인 UI

### 📊 KPI 대시보드
```
응답률 | 응답인원 | 미응답인원 | 평균만족도
```
- ✅ 4가지 핵심 지표
- ✅ 상태별 색상 (Good/Warning/Risk)
- ✅ 트렌드 화살표

### 🔍 부서 필터
```
[전체] [소속1] [소속2] [소속3]
```
- ✅ 즉시 데이터 업데이트
- ✅ Active 상태 표시
- ✅ 반응형 버튼 그룹

### 🔥 Heatmap 테이블
```
영역        | 소속1  | 소속2  | 소속3
───────────┼────────┼────────┼──────
몰입도      | 85🟢   | 72🟡   | 65🟠
조직정렬    | 78🟡   | 80🟢   | 70🟡
커리어      | 62🟡   | 58🟠   | 55🔴
```
- ✅ 점수 기반 색상 (Green/Yellow/Red)
- ✅ 범례 제공
- ✅ 호버 효과

### ⭐ **중요도 분석 (핵심 기능)**

#### 산점도 차트
```
         중요도 높음
              ↑
            ╱ ├─ 유지강화 ║ 중점개선
           ╱  │ (Green)  ║ (Red)
          ╱   ├──────────┼──────
    ─────┼───────────────┼── 만족도
         │   │           │
         │   ├─현상유지  ║ 점진개선
         │   │ (Gray)    ║ (Yellow)
         ↓
    중요도 낮음
```

- ✅ 만족도 vs 중요도 산점도
- ✅ 4가지 사분면 분석 (색상 코드)
- ✅ **요소 드래그 & 드롭** (선택 가능)
- ✅ 커스텀 Tooltip (호버 시 정보 표시)
- ✅ Reference Lines (기준선)

#### 드래그 기능
```
요소 목록
─────────────────────
📌 몰입도 (드래그 가능)
📌 조직정렬 (드래그 가능)
📌 커리어 (드래그 가능)  ← 드래그하면 차트 업데이트
📌 협업 (드래그 가능)
```

### 📈 상세 분석 패널
```
선택 요소: 커리어
```

- ✅ 선택 요소 정보 표시
- ✅ 부서별 비교 차트
- ✅ 자동 권장사항
- ✅ Empty State 처리

### 💡 주요 인사이트

## 📊 Excel 데이터 형식

### 필수 컬럼
```excel
응답자ID  소속1      소속2      소속3    직급    성별  근속년수
RES_001  마케팅팀   서울       1조     대리   남    3
RES_002  HR팀      서울       1조     과장   여    5
RES_003  개발팀     부산       2조     사원   남    1
```

### 점수 컬럼 (1-5 또는 1-100)
```excel
몰입도_Q1  몰입도_Q2  조직정렬_Q1  ...  중요도_몰입  중요도_정렬  ...
4         3         5          ...  5         4         ...
3         4         4          ...  4         5         ...
5         5         3          ...  5         4         ...
```

### 데이터 흐름
```
Excel 파일 (2025_culture_survey_basic.xlsx)
        ↓
    [Upload]
        ↓
ParseExcelFile() → 데이터 추출
        ↓
validateSurveyData() → 검증
        ↓
데이터 필터링 (소속별)
        ↓
점수 계산 (평균, 중요도 매트릭스)
        ↓
KPI + Heatmap + 차트 데이터
        ↓
📊 대시보드 렌더링
```
