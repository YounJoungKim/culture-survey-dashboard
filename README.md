# 조직문화 진단 대시보드

## 프로젝트 설명

조직문화 진단 결과를 자동으로 시각화하고 분석하는 실시간 웹 대시보드입니다. Excel 파일만 업로드하면 응답률 현황, 영역별 만족도, 중요도 분석을 한눈에 확인할 수 있습니다.

## 주요 기능

✨ **자동 데이터 처리**
- Excel RAW DATA 자동 파싱 및 검증
- 응답률, 응답 인원 자동 계산
- 영역별 점수 자동 집계

📊 **대시보드 시각화**
- KPI 카드 (응답률, 응답 인원, 평균 만족도)
- 색상 기반 Heatmap 테이블
- 반응형 레이아웃 (모바일 지원)

🔍 **중요도 분석 (핵심 기능)**
- 만족도 vs 중요도 산점도
- 사분면 분석 (중점 개선, 유지 강화 등)
- 요소 드래그 & 드롭 지원

📁 **조직 필터**
- 전체 / 소속1 / 소속2 / 소속3 선택 가능
- 필터 변경 시 전체 데이터 실시간 업데이트

💡 **상세 분석**
- 선택 요소별 부서 비교 차트
- 개선 권장사항 자동 생성
- 편차도 및 통계 정보 표시

## 기술 스택

- **Frontend**: React 18 + TypeScript
- **스타일**: Styled Components
- **차트**: Recharts
- **데이터 처리**: XLSX
- **드래그**: React DnD (선택적 확장)

## 설치 및 실행

### 1. 프로젝트 설정
```bash
cd culture_survey_dashboard
npm install
```

### 2. 개발 서버 실행
```bash
npm start
```
- 브라우저에서 `http://localhost:3000` 접속

### 3. 프로덕션 빌드
```bash
npm run build
```

## Excel 파일 형식

필요한 컬럼:
- `소속1`: 1차 부서 명 (필수)
- `소속2`: 2차 부서 명 (선택)
- `소속3`: 3차 부서 명 (선택)
- `직급`, `성별`, `근속년수`: 분석용 정보
- 각 문항별 점수 컬럼: 숫자형 (1-5점 또는 1-100점)

**파일 예시**: `2025_culture_survey_basic_rawdata.xlsx`

## 컬러 시스템

**주요 색상** (오렌지 계열)
- Primary Orange: `#FF8A00`
- Primary Light: `#FFF4E5`

**상태 색상** (고정)
- Risk (위험): `#E53935` (빨강)
- Warning (주의): `#FBC02D` (노랑)
- Good (양호): `#43A047` (초록)

## 주요 기능 상세 설명

### 1. 파일 업로드
- Drag & Drop 또는 클릭으로 업로드
- 자동 파일 검증 및 오류 표시

### 2. KPI 대시보드
- 응답률, 응답 인원, 미응답 인원, 평균 만족도
- 상태 기반 색상 표시 (좋음/주의/개선필요)

### 3. 부서 필터
- "전체" 또는 개별 부서 선택
- 선택 시 모든 차트 즉시 업데이트

### 4. Heatmap 테이블
- 행: 조직문화 영역 (몰입도, 조직정렬, 커리어 등)
- 열: 선택된 부서
- 색상: 점수 범위 기반 (초록→노랑→빨강)

### 5. 중요도 분석 차트
- X축: 만족도 (0-100점)
- Y축: 중요도 (0-100점)
- 사분면:
  - **좌상단** (중점 개선): 중요도 높음, 만족도 낮음
  - **우상단** (유지강화): 중요도 높음, 만족도 높음
  - **우하단** (점진 개선): 중요도 낮음, 만족도 높음
  - **좌하단** (현상유지): 중요도 낮음, 만족도 낮음

### 6. 상세 분석 패널
- 선택 요소의 만족도, 중요도, 편차 표시
- 부서별 비교 차트
- AI 기반 개선 권장사항
